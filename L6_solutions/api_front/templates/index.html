USD rates and sales data

This website gives an easy access to USD rates and sales data available via API from previous laboratory. Use Rates and Sales dropdowns in navigation to access desired data.

Note that the data is available for date range: 2013-01-01 - 2014-12-31


API is available at http://127.0.0.1:5000/

To access USD to PLN rate for single date use: http://127.0.0.1:5000/rates/usd/{date}

To access USD to PLN rate for date range use: http://127.0.0.1:5000/rates/usd/{start_date}/{end_date}

To access sales for single date use: http://127.0.0.1:5000/sales/{date}

To access sales for date range use: http://127.0.0.1:5000/sales/{start_date}/{end_date}
